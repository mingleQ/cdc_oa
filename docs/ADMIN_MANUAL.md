# OA 管理员操作手册

## 部门管理

进入「基础平台」，在「组织架构」区域点击「新增部门」。系统会限制删除存在子部门或用户的部门。

## 用户管理

进入「基础平台」，在「用户管理」区域可新增用户、禁用用户、重置密码。

## 角色菜单权限

进入「基础平台」的「角色与菜单权限」，点击「配置权限」，勾选角色可访问的菜单后保存。

## 流程配置

进入「基础平台」的「流程配置」，点击「新增流程版本」。

当前流程配置方式：

- 选择业务类型
- 填写流程名称
- 每行填写一个审批节点
- 保存后自动作为该业务类型的新启用流程

新提交的业务会使用新流程；历史业务保留原流程进度。

## 行车记录

进入「用车管理」，点击「登记行车记录」，填写车辆、里程、用油和归队时间。

## 公文管理

管理员可在「公文管理」中新增收文或发文，进行分发、审批、办结归档，并查看公文留痕。

## 日志审计

进入「操作日志」可查看：

- 操作日志
- 登录日志

## 备份

在服务器项目目录执行：

```bash
npm run backup
```

建议配置 Windows 任务计划程序定时执行。
