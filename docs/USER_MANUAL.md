# OA 用户操作手册

## 登录

打开系统地址，输入账号和密码登录。

## 工作台

工作台显示待处理事项、快捷申请和通知公告。

## 请假申请

1. 进入「请假管理」。
2. 点击「新增请假申请」。
3. 填写请假类型、开始日期、结束日期和事由。
4. 点击「提交审批」。
5. 在列表中点击「查看」可查看审批进度和附件。

## 出差申请

1. 进入「出差管理」。
2. 点击「新增出差申请」。
3. 填写出差类型、时间和工作事项。
4. 提交后等待审批。

## 用车申请

1. 进入「用车管理」。
2. 点击「新增用车申请」。
3. 填写用车事由、时间和车辆需求。
4. 审批通过后由管理员或负责人登记行车记录。

## 公文阅读与签收

1. 进入「公文管理」。
2. 点击公文「留痕」。
3. 可执行阅读、签收、反馈。
4. 可查看办理记录和附件。

## 附件

在申请详情或公文详情中，可以上传、预览和下载附件。

## 导出

有权限的用户可在业务列表点击「导出 Excel」。
